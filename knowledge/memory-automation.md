# 记忆自动化提取指南 (Memory Extraction Guide)

本指南供 Agent 参考，用于在任务结束后自动总结调教成果并写入记忆库。

## 识别可提取的记忆
当以下情况发生时，应提议记录记忆：
1. **解决了顽固问题**：例如修正了反复出现的画风偏移、空间畸变。
2. **发现了“魔法参数”**：某个特定的 Prompt 单词或配置项对效果有显著提升。
3. **确定了项目基调**：用户明确了全局的视觉 DNA（如：强制黑白、特定光影）。

## 提取步骤
1. **回顾对话**：从最近的对话中提取成功的 Prompt 片段。
2. **对比分析**：找出导致失败的原因和修正后的关键点。
3. **格式化**：按照 `skill-src/mangou/memories/TEMPLATE.md` 生成 Markdown。
4. **命名规范**：使用 `YYYY-MM-DD-描述性关键词.md` 命名。

## 写入位置
始终写入 `workspace/.mangou/memories/`。

## 示例总结话术
> “刚才我们通过加入 `6500K point light` 成功解决了画面过软的问题。我建议把这个经验总结到工作区的记忆模块中，以便下次自动应用。您看可以吗？”
