# 导演规则

## 分镜写法

1. 一个 `storyboards/*.yaml` 只表示一个物理镜头或一个逻辑叙事单元。
2. `content.story` 保留剧本原文，确保叙事逻辑不因概括而丢失。
3. 镜头语言描述应具有明确的物理指向：`close-up`、`medium shot`、`wide panorama` 等词级优先。

## 影视复刻与叙事匹配

1. **叙事节奏对齐**：复刻已有片段时，分镜面板应严格映射到参考视频的时间切片，确保叙事节奏（如起承转合）与原片同步。
2. **参考帧提取**：建立以秒或关键帧为单位的参考系，将抽象叙事转化为具体的视觉观测点。

## 视觉 DNA 校准

1. **视觉锚点锁定**：在编写 Prompt 前，建议明确并锁定全局视觉 DNA 参数，如：色温、曝光级别（High-key/Low-key）、材质风格策略。
2. **风格漂移控制**：严禁使用模糊的审美词汇。应使用具象的、具有物理属性的术语来定义视觉质感。
3. **标签锚定（Label Anchoring）**：使用特定项目标识（如特定编号或代号）作为跨镜头一致性的“语义锁”，通过标识的视觉复用锚定模型对物体的空间认同。
4. **真空光影物理（Vacuum Physics）**：针对深空等真空场景，强制描述“点光源效应”——极亮（如强日光）与极暗（绝对黑影）的物理对比，避开由于大气模拟生成的软阴影。
5. **关键资产一致性**：核心资产（角色、装备、标识）需具备稳定的视觉特征，并在多镜头间保持物理属性的一致性。

## 空间与叙事一致性 (Cinematic Continuity)
1. **几何脊椎 (Geometric Spine)**：
   - 针对垂直、轴对齐的大型造物（电梯、轨道、建筑），必须强制 `Central-Axis Alignment`。
   - 在所有分镜中，缆绳/轨道/结构脊柱必须处于画面与物体的正中垂直轴线上，以消除 AI 的空间定位漂移。
2. **链式叙事 (Narrative Relay)**：
   - 针对 15s 跨度的分镜，采用“接力棒（Baton）”原则。
   - 每个分镜必须包含一个从上一个分镜传承下来的视觉锚点（如 P1 特写的部件在 P2 中景中依然可见且位次正确）。
3. **时间连拍 (Temporal Burst)**：
   - 针对极高难度、物理逻辑复杂的微观动作（如重力畸变、高速撞击、穿云瞬间），采用 0.3s 物理步进。
   - 强制使用单一跟踪机位，仅改变物理属性（光影、云雾、位移），以确保视频生成模型能理解运动轨迹。
4. **动作符合度校验**：
   - 同一场景跨镜头先固定角色站位。
   - 视线、左右位次、前后关系不能乱跳，除非剧本明确写了走位。
   - 视觉校验应重点关注动作符合度、剧情逻辑以及时空背景的逻辑一致性。
